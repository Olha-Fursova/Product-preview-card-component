### Опис завдання

Основне завдання — створити цей компонент картки попереднього перегляду продукту та максимально наблизити його до дизайну.

Ваші користувачі повинні мати змогу:

- Переглядати оптимальний макет залежно від розміру екрана свого пристрою
- Переглядати стани наведення курсора та фокусування для інтерактивних елементів

### Використані технології

- HTML5 для розмітки
- CSS3 стилізація
- Flexbox
- Grid

## Висновки

Цей проєкт дуже простий, але водночас цікавий. Чому? Тому що сам дизайн дуже привабливий для ока, стилізацію елементів легко побачити навіть без файлу дизайну, пропорції та розташування елементів у контейнерах чіткі.

### Мої досягнення

В першу чергу мені було достатньо невеликої кількості часу у помірному темпі для побудови компоненту. Я знаю, що я можу виконати подібне завдання швидше, ніж за годину, але оскільки я проводила стрім при побудові та пояснювала кожен крок, то це відібрало трохи часу. Але навіть з тим, завдання було виконане трохи більше, ніж за годину. 

Я нарешті переступила через себе і використала rem замість px, тож розрахунки теж відбирали деякий час.

А також комбінування Flexbox та Grid для різного позиціонування елементів.
